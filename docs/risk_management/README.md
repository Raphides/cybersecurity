# Fundamentos de Gestão de Risco em Cybersegurança

Risco é a chance de um evento gerar um resultado não desejado. Compreender e gerenciar o risco associado a um evento é essencial para cybersegurança, em que toda ação ou omissão possui riscos associados e várias vezes nossas empresas decidem tomar esses riscos.

O objetivo é sempre melhorar a compreensão e gerenciamento do risco a partir de diversas ações.

Há alguns frameworks famosos para gestão de riscos em cybersegurança, sendo eles:
- GRC
- CSF
